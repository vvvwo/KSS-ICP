How to run the KSS-ICP:

Using "start.bat" to align input point clouds.

Instruction of input parameters:

KSS-ICP.exe PointSource.ply PointTarget.ply 

PointSource.ply: input source point cloud.
PointTarget.ply: input target point cloud.

The result is output into a .xyz format file.
